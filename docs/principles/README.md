# Founderos Principles

This directory publishes the public principles behind Founderos.

These documents exist so the repo is understandable without private context. They describe the system's governing philosophy, authority model, execution boundaries, and decision-support posture in stable public language.

They are intentionally high level. They do not include secrets, private prompts, internal operator scaffolding, or implementation-private control logic.

Use this section to understand how Founderos approaches:

- human authority
- governed action
- bounded execution
- execution handoff
- agent modes
- decision support
